Lesson One

Intro to the Web

The web is an internetwork of different computers sharing information via a standard protocol and file types

One machine will request the information from the other machine to be delivered

HTML Basics

HTML is the standard file for displaying content on the WWW

Intro to Tags

Tags in HTML tell the computer how to display the content in the file to add customization and functionality to an html document

Intro to Links

Links are imbedded connections in an html webpage that directs to another html page allowing navigation from one point to another linked one.

Adding Images

Images in HTML documents greatly enhance the functionality and aesthetics of the page

Inline vs. Block

Inline elements are read and displayed in a linear fashion, whereas Block elements are treated uniformly within the designated block formed by an invisible box allowing greater flexibility in formatting.

Lesson Two

Why start with HTML and CSS?

HTML is the basis for creating online content and CSS is the means used to avoid repitition in coding those documents

First Steps

Build a conceptual understanding of what content you're going to populate the web page with. The house analogy

HTML-CSS-DOM

HTML has syntax and rules to be followed like grammar. The browswer uses the Document Object Model to read HTML tags and translate that into a page. Cascading Style Sheets provides syntax and rules to change the way the page looks.

Boxes Everywhere!

Every element on a webpage is a box

Boxes - Grids - Rules

Thinking in terms of boxes makes life easier for the Front End Developer. It allows content to be rearranged to alter the aesthetics of the page

Boxifying Design

Start with the desired MOC and begin to draw boxes around the elemtns that comprise the end state design.
